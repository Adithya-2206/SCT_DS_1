# SkillCraft Technology - Data Science Internship

## Task 01: Data Visualization

### Objective
Create a histogram or bar chart to visualize the distribution of a continuous or categorical variable.

### Dataset
IPL Matches Dataset

### Variable Analyzed
Winning Margin (Runs)

### Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

### Result
A histogram was created to visualize the distribution of IPL match winning margins.

### Key Insights
- Most matches are won by small to moderate margins.
- Large-margin victories are less common.
- The distribution is right-skewed.
